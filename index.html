<div>
بلوک پایه برای بخش‌بندی صفحات وب. برای layout و container استفاده می‌شود و با CSS و JS تعامل دارد. بخش‌بندی و group کردن عناصر برای طراحی منظم UI.
<span>
عنصر inline برای سبک‌دهی جزئی متن یا عناصر کوچک. برای text highlight یا icon wrapper کاربرد دارد. قابلیت ترکیب با CSS و JS برای جلوه‌های خاص.
<a>
لینک برای هدایت به URL دیگر یا داخلی. با CSS hover و JS event handling برای interactivity کاربرد دارد و بخش navigation وب سایت را مدیریت می‌کند.
<img>
نمایش تصاویر در صفحات وب. Attributes: src, alt, width, height. CSS برای responsive و JS برای lazy loading یا gallery کاربرد دارد.
<ul>
لیست غیرترتیبی برای منوها و bullet list ها. CSS برای style و JS برای اضافه/حذف آیتم‌ها و interaction استفاده می‌شود.
<ol>
لیست ترتیبی برای step-by-step instructions و شماره‌گذاری. CSS و JS برای style و interaction عناصر کاربرد دارد.
<li>
آیتم‌های داخل ul یا ol. CSS برای marker, padding و hover و JS برای اضافه یا حذف آیتم‌ها کاربرد دارد.
<form>
فرم برای دریافت input کاربر. Attributes: action, method. CSS برای style و JS برای validation، Ajax و submit کاربرد دارد.
<input>
فیلد ورودی فرم‌ها: text, password, email و … JS برای validation و dynamic behavior استفاده می‌شود.
<button>
دکمه تعاملی برای trigger event یا submit فرم. CSS برای style و animation و JS برای event handling کاربرد دارد.
<header>
سرصفحه شامل لوگو، navigation و tagline. CSS برای layout و responsive و JS برای sticky menu کاربرد دارد.
<footer>
پایین صفحه شامل copyright، links یا social icons. CSS برای layout و JS برای dynamic content و interactivity کاربرد دارد.
